# 家庭健康管理 - 体检套餐管理第一版

这是家庭自用的体检套餐管理工具，当前先实现“体检”模块里的第一部分：体检套餐管理。

## 当前功能

- 上传医院公众号/小程序截图进行 OCR
- 复制截图后可直接在页面按 Ctrl+V 粘贴
- OCR 主路径使用本地 PaddleOCR 后端服务
- 区分“套餐列表截图”和“套餐详情截图”
- OCR 后进入人工核对表格
- 核对后确认进入套餐库
- 管理套餐价格、适用人群、检查项目
- 多套餐横向对比
- 管理家庭成员
- 为家人选择基础套餐并增减项目，生成体检方案
- 数据保存在当前浏览器 localStorage
- 支持导出/导入 JSON 备份数据

## 使用方式

第一次使用先安装 OCR 后端依赖：

```bat
install_backend.bat
```

之后每次使用先启动本地 OCR 服务：

```bat
start_backend.bat
```

后端启动后，再用浏览器打开 `index.html`。

默认 OCR 地址是：

```text
http://localhost:8765/api/ocr
```

Vue 和 Element Plus 当前通过 CDN 加载，所以打开前端页面时需要网络可访问 CDN。

当前 OCR 使用本地 PaddleOCR。首次运行会下载/初始化 OCR 模型，耗时会比较久；后续会快很多。

## 第一版边界

- 不做商业化和运营功能
- 不做多医院平台化
- 不做登录、账号和云同步
- 不直接信任 OCR 结果，必须人工核对后才入库
- 后端当前只负责 OCR，业务数据仍保存在浏览器 localStorage
